HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The meta tag is used to provide metadata about an HTML document.
  • ul is used for unordered lists, while li is used to end an item in the list

CSS

  • CSS provides various types of selectors to target HTML elements, such as element selectors, class selectors, ID selectors, attribute selectors, and more.
  • External CSS: A separate CSS file is linked to the HTML document using the link tag.
  • Internal CSS: CSS styles are defined within the style tags in the HTML document's head section.
  • Inline CSS: Styles are directly applied to individual HTML elements using the style attribute.
  • The CSS box model consists of content, padding, border, and margin, which collectively determine an element's dimensions and spacing.
  • Debuging css in the webbrowser is beneficial because you do not have to change and save the code.
